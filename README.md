## Overview
This project demonstrates an end-to-end **data analytics workflow**, from raw data ingestion to actionable business insights.  
It highlights practical skills in **Python, SQL, data cleaning, exploratory data analysis (EDA), dashboarding, and data storytelling** using industry-standard tools.

The objective of this project is to analyze customer shopping behavior and support data-driven decision-making.

## Dataset
- **Type:** Customer shopping behavior data (CSV)
- **Description:** Includes customer demographics, purchase details, product categories, and spending patterns
- **Status:** Cleaned and transformed for analysis

> Raw data was processed to ensure accuracy, consistency, and usability for analytics and visualization.

## Project Workflow
1. **Data Loading**
   - Loaded the dataset into Python using pandas
   - Reviewed structure and data types

2. **Exploratory Data Analysis (EDA)**
   - Generated summary statistics
   - Visualized distributions and trends
   - Identified patterns and anomalies

3. **Data Cleaning**
   - Handled missing values
   - Removed duplicates
   - Standardized column names and formats
   - Ensured data consistency

4. **SQL Analysis (MySQL)**
   - Imported cleaned data into MySQL
   - Executed analytical queries using:
     - Aggregations (SUM, AVG, COUNT)
     - GROUP BY and filtering
     - Business-focused questions

5. **Dashboard Development**
   - Built an interactive Power BI dashboard
   - Visualized KPIs and customer behavior
   - Enabled filtering by key attributes

6. **Reporting & Presentation**
   - Created a written analytical report
   - Developed a professional presentation using Gamma
   - Focused on insights, trends, and business impact
